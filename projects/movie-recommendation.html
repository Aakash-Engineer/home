Movie Recomendation System

A sophisticated content-based movie recommendation system that suggests similar movies based on textual features like plot descriptions, genres, cast, and crew information. Built with Python and deployed as an interactive web application using Streamlit.

Live GitHub



Key Features

  • Content based filtering
  • Interactive UI
  • Real-time Recommendations
  • TMDB Integration

Technical Implementation

The overall project lifecycle is divided into five key steps, Data ingestion, Model preparation, Model training, Model evaluation, Deloyment on aws through CI/CD pipelines

System Architecture

  • Data Pipeline: Custom ETL pipeline for data preprocessing and transformation
  • Model: TF-IDF vectorization with scikit-learn for text feature extraction
  • Similarity Matching: Cosine similarity for finding movie relationships
  • Data Storage: Uses Git LFS for handling large model files
