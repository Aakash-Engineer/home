End-to-end Chest Classification

This project is an end-to-end chest cancer classification system using deep learning. It follows a typical data science project life cycle, including data ingestion, data preparation, model training, and model evaluation. The goal is to classify chest CT scan images to detect signs of cancer.

Live GitHub

Key Features

  • Deployed on AWS EC2
  • Automated and image preprocessing pipeline
  • Automated CI/CD pipeline with Github Action
  • Custom CNN architecture on top of VGG16
  • Real-time inference capabilities
  • Highly flexible desing, can be extended easily

Challenge

The main challenge was to create a system with good accuracy yet using minimal computational resources.

Solution

I tried to keep the model overall size small by keeping most of the convolutional layers on freeze mode. This extensively reduced the parameters and also gave a good performing model. I used the 'imagenet' weights and excluded the top layer to simplify the architecture.

System Architecture

1. Data Ingestion

- Data collection from AWS Simple Storage Servive (S3)

- Implementation of data version control using DVC

- Automated data validation and cleaning pipeline

2. Model Preparation

- VGG16 architecture with imagenet weights

- Custom top layers for classification

- Data augmentation pipeline

3. Model Training

- Transfer learning approach with frozen layers

- MLflow for experiment tracking

- Hyperparameter optimization

4. Model Evaluation

- Comprehensive metrics tracking, accuracy and recall

- Cross-validation results

- Performance benchmarking

5. Deployment

- Containerization using Docker

- Automated CI/CD pipeline with GitHub Actions

- AWS EC2 deployment with auto-scaling

- Monitoring and logging setup

Results & Impact

  • 95% classification accuracy on test dataset
  • 50% reduction in processing time
  • Successfully deployed in AWS EC2
